# 읏차
커뮤니티 - 토이 프로젝트

# 소개 
어르신 분들을 위한 커뮤니티 개설 

# 목표
치매 예방, 우울증 예방, 정보 공유, 즐거움

# 구현 목표 기능 
1. 화상 채팅
2. 어르신들에게 유익한 영상 제공 (유튜브 추천 알고리즘) 
3. 치매 예방 게임
4. 심리 치료
5. 자유 게사판 
6. 재능 공유 

# 고려 사항
1. IT 기기에 익숙치 않기 때문에 사용이 간편해야 함
2. 연령대 선정
3. 욕설, 비매너 사용자 차단 정책 

# 개발 환경  
SPRING BOOT, JAVA, JPA, JUNIT, AWS, MSA
